Pemeriksa EXIF & Format Imej

Letakkan foto atau fail imej di bawah untuk membaca format sebenar (dikesan oleh tandatangan fail, bukan hanya sambungan), dimensi piksel, saiz fail, jenis warna dan metadata EXIF ​​seperti pembuatan dan model kamera, pendedahan, ISO dan GPS. Semuanya dihuraikan secara setempat dalam penyemak imbas anda dengan API FileReader — tiada imej pernah dimuat naik.

Privasi: imej dibaca dan dihuraikan hanya di dalam penyemak imbas anda dan tidak pernah dimuat naik ke mana-mana pelayan.

Format yang disokong dan cara pengesanan tandatangan berfungsi

Penonton imej sering meneka format daripada sambungan fail, tetapi sambungan boleh dinamakan semula kepada apa-apa sahaja. Pendekatan yang lebih dipercayai ialah membaca beberapa bait tandatangan pertama (nombor ajaib). Jadual di bawah menyenaraikan format imej utama yang dikesan oleh alat ini dan tandatangannya:

Apakah metadata EXIF ​​dan cara ia dibaca

EXIF (Format Fail Imej Boleh Tukar) ialah blok metadata yang ditulis oleh kamera dan telefon ke dalam foto, biasanya dalam JPEG, TIFF dan beberapa fail HEIC. Ia biasanya terletak dalam segmen JPEG APP1, disimpan sebagai direktori TIFF IFD yang memegang pembuatan kamera, model, masa tangkapan, pendedahan, apertur, ISO dan panjang fokus — dan kadangkala latitud dan longitud GPS. Tangkapan skrin, imej dimampatkan web dan kebanyakan PNG biasanya EXIFnya dilucutkan.

Anggaran 2026 untuk format imej dan metadata

anggaran · bukan pengukuran tangan pertama
  • Anggaran setakat 2026: JPEG kekal sebagai bekas paling biasa untuk web dan tangkapan terus keluar dari kamera, masih mendahului antara imej web.
  • Anggaran pada 2026: sejak 2017 iPhone merakam dalam HEIC secara lalai, dan model terkini kebanyakannya mengeluarkan HEIC, yang sering memerlukan transcoding pada Windows.
  • Anggaran setakat 2026: Penggunaan WebP dan AVIF dalam kalangan imej web terus meningkat, manakala kamera masih jarang menggunakannya.
  • Anggarkan pada 2026: foto telefon tunggal biasanya berukuran 2–5 MB pada 12–50 megapiksel; platform sosial biasanya memampatkan semula muat naik dan menanggalkan EXIF.

Soalan lazim

Adakah imej saya dimuat naik ke pelayan?

Tidak. Alat ini menggunakan API FileReader pelayar untuk membaca dan menghuraikan imej secara setempat; imej tidak pernah meninggalkan peranti anda dan tidak dimuat naik ke mana-mana pelayan.

Mengapa imej saya tiada EXIF?

Tangkapan skrin, imej yang dimuat turun dan dimampatkan semula daripada web dan kebanyakan PNG biasanya tidak membawa EXIF; platform sosial juga menanggalkan EXIF ​​pada muat naik untuk melindungi privasi.

Mengapa sambungan berbeza daripada format sebenar?

Sambungan hanyalah sebahagian daripada nama fail dan boleh ditukar secara bebas. Alat ini membaca bait tandatangan pada permulaan fail, jadi walaupun .png yang dinamakan semula kepada .jpg dikesan dengan betul.

Bolehkah saya melihat di mana gambar diambil?

Jika foto itu menyimpan data GPSnya (biasa untuk telefon asal yang tidak diedit), alat itu menunjukkan koordinat. Kebanyakan imej yang dimampatkan oleh platform sosial tidak lagi membawa GPS.

Adakah ia menyokong HEIC / RAW?

Alat ini mengesan HEIC/HEIF dan format yang serupa dengan tandatangan dan menunjukkan maklumat fail; sokongan penyemak imbas untuk penyahkodan HEIC adalah terhad, jadi lakaran kecil mungkin tidak dipaparkan. Untuk tontonan lancar pada Windows anda boleh menggunakan 2345 Picture Viewer.

Adakah data itu tepat?

Format tandatangan, dimensi dan medan EXIF ​​dibaca terus daripada fail dan boleh dipercayai; bahagian anggaran 2026 ialah anggaran arah aliran dan jelas dilabelkan sebagai bukan ukuran tangan pertama.
