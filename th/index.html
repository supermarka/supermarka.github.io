รูปภาพ EXIF และตัวตรวจสอบรูปแบบ

วางไฟล์รูปภาพหรือรูปภาพด้านล่างเพื่ออ่านรูปแบบจริง (ตรวจพบโดยลายเซ็นของไฟล์ ไม่ใช่แค่ส่วนขยาย) ขนาดพิกเซล ขนาดไฟล์ ประเภทสี และข้อมูลเมตา EXIF เช่น ยี่ห้อและรุ่นของกล้อง ค่าแสง ISO และ GPS ทุกอย่างจะถูกแยกวิเคราะห์ในเบราว์เซอร์ของคุณด้วย FileReader API — ไม่มีการอัพโหลดรูปภาพเลย

วางรูปภาพที่นี่ หรือคลิกเพื่อเลือก

ความเป็นส่วนตัว: รูปภาพจะถูกอ่านและแยกวิเคราะห์ภายในเบราว์เซอร์ของคุณเท่านั้น และจะไม่มีการอัปโหลดไปยังเซิร์ฟเวอร์ใดๆ

รูปแบบที่รองรับและวิธีการตรวจสอบลายเซ็นทำงาน

ผู้ดูรูปภาพมักจะเดารูปแบบจากนามสกุลไฟล์ แต่นามสกุลสามารถเปลี่ยนชื่อเป็นอะไรก็ได้ แนวทางที่เชื่อถือได้มากกว่าคือการอ่านไบต์ลายเซ็นสองสามตัวแรก (ตัวเลขมหัศจรรย์) ตารางด้านล่างแสดงรายการรูปแบบรูปภาพหลักที่เครื่องมือนี้ตรวจพบและลายเซ็น:

รูปแบบลายเซ็นไฟล์ (ฐานสิบหก)การใช้งานทั่วไป

ข้อมูลเมตา EXIF คืออะไรและอ่านอย่างไร

EXIF (รูปแบบไฟล์ภาพที่แลกเปลี่ยนได้) คือบล็อกของข้อมูลเมตาที่กล้องและโทรศัพท์เขียนลงในภาพถ่าย โดยทั่วไปจะเป็นไฟล์ JPEG, TIFF และ HEIC บางไฟล์ โดยปกติจะอยู่ในเซ็กเมนต์ JPEG APP1 ซึ่งจัดเก็บเป็นไดเร็กทอรี TIFF IFD ที่เก็บยี่ห้อ รุ่น เวลาถ่ายภาพ ค่ารับแสง ค่ารูรับแสง ISO และทางยาวโฟกัส และบางครั้งก็มี GPS ละติจูดและลองจิจูด ภาพหน้าจอ รูปภาพที่บีบอัดจากเว็บ และ PNG ส่วนใหญ่มักจะไม่มี EXIF ​​​​

ประมาณการปี 2026 สำหรับรูปแบบรูปภาพและข้อมูลเมตา

การประมาณค่า · ไม่ใช่การวัดโดยตรง
  • ประมาณการในปี 2026: JPEG ยังคงเป็นคอนเทนเนอร์ที่ใช้บ่อยที่สุดสำหรับเว็บและช็อตที่ถ่ายจากกล้องโดยตรง โดยยังคงเป็นผู้นำในบรรดารูปภาพบนเว็บ
  • ประมาณการ ณ ปี 2026: ตั้งแต่ปี 2017 iPhone จะถ่ายภาพในรูปแบบ HEIC ตามค่าเริ่มต้น และรุ่นล่าสุดจะส่งออกข้อมูลเป็น HEIC เป็นหลัก ซึ่งมักต้องมีการแปลงรหัสบน Windows
  • ประมาณการในปี 2026: การใช้ WebP และ AVIF ในภาพเว็บเพิ่มขึ้นเรื่อยๆ ในขณะที่กล้องยังไม่ค่อยได้ใช้งาน
  • ประมาณการ ณ ปี 2026: ภาพถ่ายในโทรศัพท์ภาพเดียวโดยทั่วไปจะมีขนาด 2–5 MB ที่ 12–50 ล้านพิกเซล; แพลตฟอร์มโซเชียลมักจะบีบอัดการอัปโหลดอีกครั้งและตัด EXIF

คำถามที่พบบ่อย

รูปภาพของฉันถูกอัพโหลดไปยังเซิร์ฟเวอร์หรือไม่?

ไม่ เครื่องมือนี้ใช้เบราว์เซอร์ FileReader API เพื่ออ่านและแยกวิเคราะห์รูปภาพในเครื่อง รูปภาพจะไม่ออกจากอุปกรณ์ของคุณและไม่ได้อัปโหลดไปยังเซิร์ฟเวอร์ใด ๆ

ทำไมรูปภาพของฉันถึงไม่มี EXIF

ภาพหน้าจอ รูปภาพที่ดาวน์โหลดและบีบอัดซ้ำจากเว็บ และ PNG ส่วนใหญ่มักจะไม่มี EXIF แพลตฟอร์มโซเชียลยังตัด EXIF ​​ในการอัปโหลดเพื่อปกป้องความเป็นส่วนตัว

เหตุใดส่วนขยายจึงแตกต่างจากรูปแบบจริง

นามสกุลเป็นเพียงส่วนหนึ่งของชื่อไฟล์และสามารถเปลี่ยนแปลงได้อย่างอิสระ เครื่องมือนี้จะอ่านไบต์ลายเซ็นที่จุดเริ่มต้นของไฟล์ ดังนั้นแม้แต่ .png ที่เปลี่ยนชื่อเป็น .jpg ก็ยังตรวจพบได้อย่างถูกต้อง

ฉันขอดูได้ไหมว่าถ่ายรูปที่ไหน?

หากภาพถ่ายเก็บข้อมูล GPS ไว้ (โดยทั่วไปสำหรับต้นฉบับของโทรศัพท์ที่ไม่มีการตัดต่อ) เครื่องมือจะแสดงพิกัด รูปภาพส่วนใหญ่ที่บีบอัดโดยแพลตฟอร์มโซเชียลไม่มี GPS อีกต่อไป

รองรับ HEIC / RAW หรือไม่

เครื่องมือตรวจจับ HEIC/HEIF และรูปแบบที่คล้ายกันด้วยลายเซ็นและแสดงข้อมูลไฟล์ การสนับสนุนเบราว์เซอร์สำหรับการถอดรหัส HEIC นั้นมีจำกัด ดังนั้นภาพขนาดย่ออาจไม่แสดงผล เพื่อการรับชมที่ราบรื่นบน Windows คุณสามารถใช้ 2345 Picture Viewer

ข้อมูลถูกต้องหรือไม่?

ลายเซ็นรูปแบบ ขนาด และฟิลด์ EXIF ​​ถูกอ่านโดยตรงจากไฟล์และเชื่อถือได้ ส่วนการประมาณการปี 2026 เป็นการประมาณการแนวโน้มและมีป้ายกำกับชัดเจนว่าไม่ใช่การวัดโดยตรง
