Supported formats and how signature detection works

Image viewers often guess the format from the file extension, but an extension can be renamed to anything. A more reliable approach is to read the first few signature bytes (the magic number). The table below lists the main image formats this tool detects and their signatures:

What EXIF metadata is and how it is read

EXIF (Exchangeable Image File Format) is a block of metadata that cameras and phones write into a photo, commonly in JPEG, TIFF and some HEIC files. It usually sits in the JPEG APP1 segment, stored as TIFF IFD directories holding the camera make, model, capture time, exposure, aperture, ISO and focal length — and sometimes GPS latitude and longitude. Screenshots, web-compressed images and most PNGs usually have their EXIF stripped.

Frequently asked questions

Are my images uploaded to a server?

No. The tool uses the browser FileReader API to read and parse images locally; the image never leaves your device and is not uploaded to any server.

Why does my image have no EXIF?

Screenshots, images downloaded and re-compressed from the web, and most PNGs usually carry no EXIF; social platforms also strip EXIF on upload to protect privacy.

Why is the extension different from the real format?

An extension is just part of the file name and can be changed freely. This tool reads the signature bytes at the start of the file, so even a .png renamed to .jpg is detected correctly.

Can I see where a photo was taken?

If the photo kept its GPS data (common for unedited phone originals), the tool shows the coordinates. Most images compressed by social platforms no longer carry GPS.

Does it support HEIC / RAW?

The tool detects HEIC/HEIF and similar formats by signature and shows file info; browser support for decoding HEIC is limited, so a thumbnail may not render. For smooth viewing on Windows you can use 2345 Picture Viewer.

Is the data accurate?

Format signatures, dimensions and EXIF fields are read directly from the file and are reliable; the 2026 estimates section is a trend estimate and is clearly labeled as not first-hand measurement.
