EXIF Gambar & Pemeriksa Format

Letakkan file foto atau gambar di bawah untuk membaca format aslinya (terdeteksi berdasarkan tanda tangan file, bukan hanya ekstensi), dimensi piksel, ukuran file, jenis warna, dan metadata EXIF seperti merek dan model kamera, eksposur, ISO, dan GPS. Semuanya diuraikan secara lokal di browser Anda dengan FileReader API — tidak ada gambar yang diunggah.

Format yang didukung dan cara kerja deteksi tanda tangan

Pemirsa gambar sering kali menebak format dari ekstensi file, namun ekstensi dapat diubah namanya menjadi apa saja. Pendekatan yang lebih dapat diandalkan adalah dengan membaca beberapa byte tanda tangan pertama (angka ajaib). Apa itu metadata EXIF dan cara membacanya

EXIF (Exchangeable Image File Format) adalah blok metadata yang ditulis kamera dan ponsel ke dalam foto, biasanya dalam format JPEG, TIFF, dan beberapa file HEIC. Biasanya berada di segmen JPEG APP1, disimpan sebagai direktori TIFF IFD yang menyimpan merek kamera, model, waktu pengambilan, pencahayaan, bukaan, ISO, dan panjang fokus — dan terkadang garis lintang dan bujur GPS. Tangkapan layar, gambar yang dikompresi web, dan sebagian besar PNG biasanya EXIF-nya dihilangkan.

Perkiraan tahun 2026 untuk format gambar dan metadata

perkiraan · bukan pengukuran langsung
  • Perkiraan pada tahun 2026: JPEG tetap menjadi wadah paling umum untuk web dan pengambilan gambar langsung dari kamera, masih memimpin di antara gambar web.
  • Perkiraan pada tahun 2026: sejak tahun 2017, iPhone memotret dalam HEIC secara default, dan model terbaru sebagian besar menghasilkan HEIC, yang sering kali memerlukan transcoding pada Windows.
  • Perkiraan pada tahun 2026: Adopsi WebP dan AVIF pada gambar web terus meningkat, sementara kamera masih jarang menggunakannya.
  • Perkiraan pada tahun 2026: satu foto ponsel biasanya berukuran 2–5 MB pada 12–50 megapiksel; platform sosial biasanya mengompres ulang unggahan dan menghapus EXIF.

Pertanyaan yang sering diajukan

Apakah gambar saya diunggah ke server?

Tidak. Alat ini menggunakan API FileReader browser untuk membaca dan mengurai gambar secara lokal; gambar tidak pernah meninggalkan perangkat Anda dan tidak diunggah ke server mana pun.

Mengapa gambar saya tidak memiliki EXIF?

Tangkapan layar, gambar diunduh dan dikompres ulang dari web, dan sebagian besar PNG biasanya tidak membawa EXIF; platform sosial juga menghapus EXIF ​​pada unggahan untuk melindungi privasi.

Mengapa ekstensi berbeda dari format sebenarnya?

Ekstensi hanyalah bagian dari nama file dan dapat diubah dengan bebas. Alat ini membaca byte tanda tangan di awal file, sehingga .png yang diubah namanya menjadi .jpg pun terdeteksi dengan benar.

Bisakah saya melihat di mana foto itu diambil?

Jika foto menyimpan data GPS-nya (umumnya terdapat pada ponsel asli yang belum diedit), alat akan menunjukkan koordinatnya. Sebagian besar gambar yang dikompresi oleh platform sosial tidak lagi membawa GPS.

Apakah ini mendukung HEIC/RAW?

Alat ini mendeteksi HEIC/HEIF dan format serupa dengan tanda tangan dan menampilkan info file; dukungan browser untuk mendekode HEIC terbatas, sehingga thumbnail mungkin tidak ditampilkan. Untuk kelancaran tampilan di Windows Anda dapat menggunakan 2345 Picture Viewer.

Apakah datanya akurat?

Tanda tangan format, dimensi, dan bidang EXIF ​​dibaca langsung dari file dan dapat diandalkan; bagian perkiraan tahun 2026 merupakan perkiraan tren dan diberi label jelas sebagai bukan pengukuran langsung.
